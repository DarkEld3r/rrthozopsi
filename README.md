## Overview

Simple experimental game.

## Building

### Instal SDL2
Follow instructions in the README file:

https://github.com/AngryLawyer/rust-sdl2

### Build
```sh
git clone https://github.com/DarkEld3r/rrthozopsi
cd rrthozopsi
cargo build
```
## Running

```sh
cargo run
```
